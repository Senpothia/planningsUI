Labplan: une application web pour faciliter la gestion de vos essais en laboratoire

Après plusieurs années d'activité dans le test et contrôle de conformité de produits électroniques, j'ai eu besoin de me doter d'un outil fiable et pratique. Aucune application sur le marché ne répondait précisement à mes besoins à un coût acceptable. Par conséquent, j'ai entrepris d'utiliser mes compétances en programmation web pour développer ma propre application taillée sur mesure.

Principe de fonctionnement:

• L'application permet la gestion des quailfications de produits électroniques
• Vous devez créer un compte utilisateur.
• Une qualification se compose d'une suite d'essais.
• Un essai est la réalisation d'une procédure de test.
• Un test concerne plusieurs échantillons
• Un test peut comprendre plusieurs séquences
• Un échantillon peut intervenir dans plusieurs essais et plusieurs séquences.

Mise en oeuvre:

• Créer une qualification à partir du formulaire dédié
• Créer ou utiliser les essais déjà définis pour définir le contenu de la qualification.
• Créer les échantillons de test à partir du formulaire associé.
• Créer les séquences de test en associant les échantillons
• Compléter les séquences de test avec les résultats observés au cours des essais
• Vous pouvez modifier les éléments de la quaification à loisir
